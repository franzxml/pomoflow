# Pomoflow

## Deskripsi
Proyek ini adalah aplikasi *Pomodoro Timer* berbasis web yang dirancang untuk membantu meningkatkan produktivitas dan fokus pengguna. Aplikasi ini memungkinkan pengguna untuk mengatur waktu kerja dan istirahat secara sistematis dengan antarmuka yang bersih dan intuitif, serta mendukung fitur notifikasi dan navigasi yang responsif.

## Teknologi
- HTML5 untuk struktur halaman
- CSS3 untuk gaya dan tata letak
- *Vanilla JavaScript* untuk logika timer dan interaksi
- Modul JavaScript ES6 untuk pengorganisasian kode yang rapi

## Struktur Folder
```
root/
│── index.html
│── README.md
└── assets/
    ├── css/
    │   └── style.css
    └── js/
        ├── main.js
        └── modules/
            ├── navigation.js
            ├── notifications.js
            └── timer.js
```

## Cara Menjalankan
1. Unduh atau *clone* proyek
2. Buka folder proyek
3. Jalankan file `index.html` di browser
4. Jika ingin *live reload*, gunakan ekstensi *Live Server* di VS Code
